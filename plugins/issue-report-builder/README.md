# Issue Report Builder（研发反馈单生成器）

把随口描述的问题/需求整理成规范反馈单（Markdown + assets），让研发团队可直接处理。

## 功能

- 自动用 Chrome DevTools 复现操作、采集截图、控制台报错和网络请求
- 遇登录页主动请用户登录，确实采集不到的才引导用户手动补充
- Bug 单：产出操作步骤、输入参数、截图、控制台错误、后端日志、环境信息
- 需求单：产出需求描述、上下文，主动质疑是否为产品级通用需求

## 触发词

当用户说「这个功能有bug」「页面报错了」「帮我写个 bug 报告」「提个需求」「整理一个问题反馈单」等时触发。
